# 📍 Conecta Bairro

Um sistema completo de gestão de eventos comunitários.

### 🚀 Tecnologias
- **Backend:** Python com FastAPI e SQLModel (SQLite).
- **Frontend:** HTML5, CSS3, JavaScript e Bootstrap 5.

### 🛠️ Como rodar
1. Instale as dependências: `pip install fastapi sqlmodel uvicorn`
2. Inicie o servidor: `uvicorn main:app --reload`
3. Abra o `index.html` no navegador.
